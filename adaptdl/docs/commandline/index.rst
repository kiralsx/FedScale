AdaptDL Command-line Interface
==============================

The following pages describe how to use AdaptDL CLI to submit and manage jobs
in an AdaptDL-scheduled cluster. If you are instead interested in training in a
standalone setting, see :doc:`Standalone Training<../standalone-training>`.

.. note::

   The following instructions assume you have already installed the AdaptDL
   scheduler on Kubernetes. If you have not, see :doc:`Installation
   <../installation/index>`.
